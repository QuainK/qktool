## [3.1.0] - 2020.04.20

### 增加

  - PC 端页面宽度 >= 700px

- 移动端导航栏增加展开收起按钮（汉堡包按钮）

### 改动

- vue-router 模式从 history 改成 hash

- 更换主色为黑白两色
  
- 一些项目的内容样式稍作修改

- 首页头像阴影增大，形状从圆角矩形改成圆形

### 修复

- 修复移动端页面右边会溢出的问题。

  > 原因是导航栏边距过大，现已缩小

- 修复输入错误路径并没有转至 404 页面，而是转至首页的问题。

  > vue-router 模式从 history 改成 hash

## [3.0.0]- 2020.04.14

### 增加

- 增加主要颜色 浅绿 #75ec66

- 增加欢迎页，并作为首页

- 增加笔记页

### 改动

- **更换 git 仓库**

  > 因为工程文件结构不同，3.0.0 版本及更新版本的工程使用 @vue/cli 搭建，而旧版本使用原生 HTML/CSS/JS，所以两者存放在不同的 git 仓库。
  >
  > 版本号表示的是整个网站的版本号，继续增加并不重新开始，git 仓库使用 tag 标记版本号。

  > 2.0.0 及以下：QKTool 改成 QKTool_2019
  >
  > 3.0.0 及以上：新建 qktool

- 使用 Vue.js 重构工程，根据页面布局解耦成 Vue 组件，并使用路由懒加载

- 导航栏“更新日志”改为“关于”，原“更新日志”放在“关于”内

### 修复

- 修复“关于”页主体和顶部底部因边距导致背景颜色不连贯的问题。

  > 主体 main 和顶部 header、底部 footer 从外边距 margin 改用内边距 padding

### 移除

- 移除较为复杂、资源较多的子网页/子项目。目前包括但不限于：

  - 风信子 Hyacinth

  - 管理系统演示 Management System Demo

  > 这些项目不放在此工程中，分别单独建立工程项目和 git 仓库。
  >
  > 部署时，它们的编译后部署文件仍然放在此工程的目录中。
